# Mio's blog website

## 介绍

Mio 的个人 blog 项目，使用 Next.js 框架搭建，并且使用了 Vercel 的 serverless 部署能力。

组件皆为手搓！！！

## 线上地址

[Mio's blog](https://mio-blog.vercel.app)

## 二次开发

你可以 clone 下来，进行二次开发。

环境：
- Node.js: v22
- pnpm: v9+

1. 安装依赖
```bash
pnpm install
```
2. 本地启动
```bash
pnpm dev
```

然后打开 [http://localhost:3000](http://localhost:3000) 即可。
